## Run Locally

Clone the project

```bash
  git clone https://github.com/sashad2003/users-review-app.git
```

Go to the project directory

```bash
  cd users-review-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Tech Stack

**Client:** React, Vite
